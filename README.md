# HebLib — Твой Академический Словарь 📚🇮🇱

**HebLib** (Hebrew Library) — это персональное веб-приложение для систематизации изучения иврита. В отличие от обычных словарей, оно позволяет создавать собственную базу знаний с учетом грамматических особенностей языка.

## 🚀 Основные фишки
- **Smart Translate (✨):** Мгновенный перевод слов через API.
- **Deep Integration:** Быстрый переход к экспертным базам данных (**Reverso Context** для транскрипции и **Pealim** для грамматических таблиц).
- **Grammar-First:** Поля для фиксации рода, корня и множественного числа.
- **PWA Ready:** Приложение устанавливается на iPhone/Android как обычная программа.
- **Privacy:** Все данные хранятся только на твоем устройстве (LocalStorage).

## 🛠 Технологии
- HTML5 / CSS3 (Адаптивный дизайн под iOS)
- Vanilla JavaScript
- MyMemory API (Перевод)
