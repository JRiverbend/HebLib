# HebLib — Hebrew Academic Library 📚🇮🇱

**HebLib** — это персональный цифровой словарь для глубокого изучения иврита. Проект создан как академическое дополнение к экосистеме [FunHebrew](https://jriverbend.github.io/FunHebrew/).

## Основная концепция
В отличие от обычных словарей, HebLib ориентирован на **активное обучение**. Пользователь не просто ищет перевод, а самостоятельно формирует базу знаний, систематизируя слова по грамматическим признакам.

## Функциональные возможности
- **Категоризация:** разделение слов на существительные, глаголы и прилагательные с соответствующими полями (корень, род, число).
- **Транскрипция и контекст:** возможность записи звучания и примеров предложений для лучшего запоминания.
- **Интеллектуальный интерфейс:** адаптивные формы ввода, которые меняются в зависимости от части речи.
- **Аудио-поддержка:** встроенная озвучка слов на иврите.
- **PWA (Progressive Web App):** приложение можно установить на экран смартфона как нативную программу.

## Технологический стек
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (Vanilla JS).
- **Storage:** LocalStorage для полной приватности данных (все данные хранятся только на устройстве пользователя).
- **Deployment:** GitHub Pages.
