# HebLib — Hebrew Academic Library 📚🇮🇱

**HebLib** — это персональный цифровой инструмент для систематизации и углубленного изучения иврита. Проект разработан как академическое дополнение к экосистеме [FunHebrew](https://jriverbend.github.io/FunHebrew/).

## Основная концепция
В отличие от массовых словарей, HebLib ориентирован на **активное обучение**. Пользователь формирует персональную базу знаний, самостоятельно фиксируя грамматические признаки слов, что способствует лучшему запоминанию и пониманию логики языка.

## Функциональные возможности
- **Интегрированные лингвистические мосты:** Прямой доступ к базам Reverso Context и Pealim для верификации транскрипции и грамматических форм.
- **Систематизация данных:** Фиксация рода, множественного числа и контекста использования.
- **Интеллектуальный помощник:** Функция автоматического перевода (Smart Translate) для ускорения ввода данных.
- **Архитектура PWA:** Возможность установки на мобильные устройства как нативного приложения.
- **Локальное хранение:** Полная приватность данных. База хранится в LocalStorage устройства пользователя.

## Технологический стек
- **Frontend:** HTML5, CSS3 (Modern UI/iOS optimized), JavaScript (Vanilla JS).
- **API Integration:** MyMemory API для лингвистических запросов.
- **Deployment:** GitHub Pages.
